Способи підключення стилів

Селектори CSS — це інструмент, що дозволяє вибирати елементи HTML для застосування до них стилів. Вони є ключовим елементом у створенні стильового оформлення веб-сторінки.
Основні типи селекторів:
1. Селектори елементів (типу) Вибирають усі елементи певного типу, наприклад, всі абзаци `

` або всі заголовки `

`.
2. Класові селектори Вибирають елементи з певним значенням атрибута `class`. Можуть використовуватися для групування кількох елементів із подібним стилем.
3. Ідентифікатори (селектори id) Використовуються для вибору конкретного елемента за унікальним атрибутом `id`. Призначені для вибору одного елемента на сторінці.
4. Групові селектори Дозволяють застосовувати одні й ті ж стилі до кількох елементів, перераховуючи їх через кому.
5. Комбінатори селекторів Визначають залежності між елементами (наприклад, дочірні елементи, сусідні тощо). Включають такі види:
- Дочірній селектор: вибирає елементи, які є прямими дочірніми.
- Суміжний селектор: вибирає елемент, який знаходиться безпосередньо після іншого.
- Загальний сусідній селектор: вибирає всі елементи після певного.
6. Атрибутні селектори Вибирають елементи з певними атрибутами (або їхніми значеннями).
7. Псевдокласи Використовуються для вибору елементів у певному стані (наприклад, наведення курсора, активний лінк тощо).
8. Псевдоелементи Дозволяють стилізувати певні частини елемента, наприклад, першу літеру або вставити контент до чи після елемента.
Особливості:
- Специфічність: визначає пріоритетність селекторів: селектори `id` мають вищий пріоритет, ніж класові, а класові — вищий за селектори елементів.
- Наслідування: деякі стилі передаються дочірнім елементам автоматично (наприклад, колір тексту), але інші — ні (наприклад, відступи).
Селектори забезпечують гнучкість і контроль у стильовому оформленні, дозволяючи організовувати структуру CSS для легшої підтримки.

Селектори

Селектори CSS — це частина синтаксису CSS, яка використовується для вибору
HTML-елементів, до яких будуть застосовуватись стилі. Вони визначають, які
саме елементи сторінки слід стилізувати.

1. Селектори елементів (типу)
2. Класові селектори

4. Групові селектори
5. Комбінатори селекторів
6. Атрибутні селектори

Селектор тегу

Селектор тегу (або селектор типу) використовується для вибору всіх HTML-елементів певного типу.

Приклад: стилізуємо всі абзаци 'p' і заголовки 'h1' на сторінці.

Код: 

            <style>
             p {
             color: blue;
             font-size: 16px;
             }
              h1 {
             text-align: center;
             font-weight: bold;
              }
             </style>

              <p>Це приклад абзацу.</p>
              <h1>Це приклад заголовка.</h1>

Селектор класу

Селектор класу використовується для вибору елементів із певним атрибутом class. Перед назвою класу в CSS використовується крапка (.).

Приклад: стилізуємо елементи з класом highlight.

Код: 


              <style>
             .highlight {
             background-color: yellow;
             font-style: italic;
            }
            </style>
            <p class="highlight">Цей текст виділено.</p>
            <div class="highlight">Це виділений блок.</div>

Селектор ідентифікатор

Селектор ідентифікатор використовується для вибору елемента з унікальним атрибутом id. Перед назвою ідентифікатора в CSS використовується символ решітки (#).

Приклад: стилізуємо елемент із ідентифікатором main-title.

Крд: 

                
                <style>
                #main-title {
                  color: red;
                  font-size: 24px;
                  text-transform: uppercase;
                }
                </style>
                <h1 id="main-title">Це головний заголовок.</h1>

Інші селектори

Самостійна робота №2

1. Для чого використовуються каскадні таблиці стилів?

Каскадні таблиці стилів (CSS) використовуються для опису зовнішнього вигляду веб-сторінок, зокрема 
 кольорів, шрифтів, розмірів, відступів тощо. Вони дозволяють створювати стильове оформлення для HTML-документів, забезпечуючи гармонійний вигляд і узгодженість елементів на сторінці.



2. Які методи визначення стилів HTML-документі ви знаєте?

Існують три основні методи визначення стилів у HTML-документі:  
- Вбудовані стилі (inline styles): стиль вказується безпосередньо в атрибуті <style> HTML-елемента. 
 
- Внутрішні стилі (internal styles): стиль визначається в секції <style> всередині <head> HTML-документа. 
 
- Зовнішні стилі (external styles): стиль визначається у окремому CSS-файлі, який підключається 
за допомогою тега <link>.



3. У чому різниця між принципами каскадування і спадкування?

- Каскадування визначає, як стилизуються елементи, коли стилі застосовуються одночасно з кількох джерел. 
Стиль з більшим рівнем специфічності або пізніше вказаний стиль замінює попередній.
- Спадкування описує процес, коли дочірні елементи автоматично отримують стилі від батьківських 
елементів, якщо для них не задано своїх власних стилів.



4. Як в HTML-документах оформлюються селектори тегу і клас?

- Селектор тегу вибирає всі елементи певного типу, наприклад, всі абзаци або заголовки на сторінці.  
- Селектор класу вибирає елементи з певним значенням атрибута <class>, що дозволяє застосовувати 
стилі до кількох елементів з однаковим класом.



5. Для чого використовується ідентифікатор?

Ідентифікатор (<id>) використовується для унікальної ідентифікації елемента на сторінці.
 Кожен <id> повинен бути унікальним, і він дозволяє застосовувати специфічні стилі чи працювати з елементами за допомогою JavaScript.



6. Навести приклади оформлення текстових елементів.

Оформлення текстових елементів включає зміну кольору, шрифтів, вирівнювання тексту, розміру шрифтів, стилю
 (наприклад, курсив або жирний) та інших параметрів, що дозволяють контролювати вигляд тексту на веб-сторінці.

Шрифти. Google fonts

Шрифти в HTML
У HTML шрифти використовуються для визначення зовнішнього вигляду тексту на веб-сторінці. Для цього застосовуються стилі CSS, через
які можна вказати різні параметри шрифтів: їх тип, розмір, товщину, стиль, колір тощо.


Основні властивості шрифтів в CSS:
1. font-family — визначає тип шрифту.
2. font-size — задає розмір шрифту.
3. font-weight — визначає товщину шрифтів (наприклад, bold для жирного шрифта).
4. font-style — встановлює стиль шрифта (наприклад, italic для курсиву).
5. line-height — налаштовує міжрядковий інтервал.
6. letter-spacing — дозволяє регулювати відстань між буквами.


Приклад використання шрифтів у CSS:

  <style>

    body {

        font-family: Arial, sans-serif;

        font-size: 16px;

        color: #333;

    }

    h1 {

        font-family: 'Georgia', serif;

        font-size: 24px;

        font-weight: bold;

    }

</style>


<body>

    <h1>Заголовок</h1>

    <p>Це параграф тексту, оформлений стандартним шрифтом.</p>

</body>


Google Fonts
Google Fonts — це безкоштовний онлайн-сервіс, який надає велику бібліотеку шрифтів для використання на веб-сторінках. Він дозволяє веб-розробникам легко додавати шрифти без необхідності завантажувати їх на сервери.

Як використовувати Google Fonts:
1. Вибираєте шрифт на сайті Google Fonts.
2. Копіюєте посилання на шрифт або CSS-код для підключення.
3. Додаєте посилання в розділ <head> вашого HTML-документа.
4. Використовуєте шрифт у CSS, вказуючи його в властивості font-family.

Приклад використання Google Fonts:
1. Перейдіть на Google Fonts і виберіть шрифт, наприклад, "Roboto".
2. Скопіюйте та додайте цей код у тег <head> вашого HTML-документа:
<head>
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
</head>

3. Використовуйте шрифт у CSS: <style>
body {
font-family: 'Roboto', sans-serif;
}
h1 {
font-weight: 700;
}
</style>

Переваги використання Google Fonts:
- Широкий вибір: сотні різних шрифтів, які підходять для різних стилів і мов.
- Швидкість: шрифти завантажуються з серверів Google, що гарантує їх високу доступність і швидкість завантаження.
- Легкість використання: не потрібно завантажувати файли шрифтів на сервер, що спрощує їх підключення.

Стилізація таблиць в CSS
Таблиці в HTML можна стилізувати за допомогою CSS для покращення їх зовнішнього вигляду та структури.
Стилізація таблиць включає в себе налаштування таких елементів, як межі, відступи, вирівнювання
тексту та кольори фону. CSS дозволяє контролювати вигляд усіх частин таблиці: рядків, стовпців,
заголовків і клітинок.


Основні властивості для стилізації таблиць:
1. border — задає межі таблиці, рядків або клітинок.
2. border-collapse — визначає, чи повинні межі таблиці бути злиті (одна для всіх клітинок) чи окремі.
3. padding — встановлює внутрішні відступи в клітинках таблиці.
4. text-align — вирівнює текст у клітинках таблиці (наприклад, ліворуч, по центру, праворуч).
5. background-color — встановлює колір фону для клітинок або рядків.
6. color — визначає колір тексту в клітинках.

Приклад стилізації таблиці:

  <style>

    table {

        width: 100%;

        border-collapse: collapse;

    }

    th, td {

        border: 1px solid #ddd;

        padding: 8px;

        text-align: left;

    }

    th {

        background-color: #f2f2f2;

        font-weight: bold;

    }

    tr:nth-child(even) {

        background-color: #f9f9f9;

    }

</style>

Списки в HTML можуть бути маркованими (ul), нумерованими (ol) або визначеними (dl). CSS дозволяє змінювати вигляд

Основні властивості для стилізації списків:
1. list-style-type — визначає тип маркера для маркованих списків (наприклад, круглий, квадратний або ромбічний).

Висновок про стильове оформлення та селектори
Аналізуючи матеріал, що ми розглядали, я розумію, наскільки важливим є освоєння теми стильового оформлення в HTML і CSS. Засвоєння основ каскадних таблиць стилів (CSS) дає мені можливість не тільки змінювати вигляд веб-сторінок, а й створювати їх візуально привабливими та зручними для користувачів.


Вивчення селекторів дозволяє точніше і гнучкіше визначати, як і де застосовуються стилі на сторінці. Зрозуміти різницю між селекторами тегів, класів та ідентифікаторів важливо для того, щоб ефективно працювати з елементами на веб-сторінці. Селектори дають можливість вибирати елементи з точністю до кожного тегу або групи елементів, що дозволяє створювати індивідуальні стилі для різних частин сторінки без дублювання коду.


Також важливим аспектом є розуміння принципів каскадування та спадкування в CSS. Це дозволяє ефективно керувати стилями, коли на сторінці є кілька джерел стилів або елементи успадковують стилі від своїх батьків.


Використання зовнішніх ресурсів, таких як Google Fonts, відкриває можливості для вибору красивих та універсальних шрифтів для веб-дизайну без потреби завантажувати їх на сервери. Це значно спрощує роботу та дозволяє створювати професійно виглядаючі сторінки.


Загалом, я розумію, що стильове оформлення та правильне використання селекторів — це важливі навички
для кожного веб-розробника. Вони дозволяють створювати гармонійні, зручні та стильні веб-сторінки, які
відповідають вимогам сучасного дизайну і забезпечують комфортне сприйняття інформації користувачами.

function createArray(size) {
  const array = [];
  for (let i = 0; i < size; i++) {
    array.push(Math.floor(Math.random() * 100)); // Заповнюємо випадковими числами від 0 до 99
  }
  return array;
}

// Функція для пошуку максимального серед парних елементів
function findMaxEvenElement(array) {
  let maxEven = -Infinity;
  for (let num of array) {
    if (num % 2 === 0 && num > maxEven) {
      maxEven = num;
    }
  }
  return maxEven;
}

// Функція для пошуку мінімального серед елементів з парними індексами
function findMinAtEvenIndex(array) {
  let minAtEvenIndex = Infinity;
  for (let i = 0; i < array.length; i += 2) {
    if (array[i] < minAtEvenIndex) {
      minAtEvenIndex = array[i];
    }
  }
  return minAtEvenIndex;
}

// Функція для заміни елементів у масиві
function swapElements(array, element1, element2) {
  const index1 = array.indexOf(element1);
  const index2 = array.indexOf(element2);
  if (index1 !== -1 && index2 !== -1) {
    [array[index1], array[index2]] = [array[index2], array[index1]];
  }
}

// Функція сортування методом вставки
function insertionSort(array) {
  for (let i = 1; i < array.length; i++) {
    let key = array[i];
    let j = i - 1;
    while (j >= 0 && array[j] > key) {
      array[j + 1] = array[j];
      j--;
    }
    array[j + 1] = key;
  }
}
